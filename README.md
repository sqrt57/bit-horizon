# BitHorizon

An Android-native incremental game utilizing cascading mathematical production loops and dark-terminal aesthetics. Infinite scaling, finite UI boundaries.

- **Package:** `com.cosmic.bithorizon`
- **Platform:** Android (min SDK 26)
- **UI:** Jetpack Compose (Material 3)

See [design/gdd.md](design/gdd.md) for the full game design document.
